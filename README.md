## Opchannel Boilerplate

### 首先安装node

https://nodejs.org/en/
按照指示安装完node

### clone repo
$ git clone https://github.com/zhaogaolei/lz-webpack-angularjs.git

### 安装依赖包
$ npm install

### 运行
$ npm start

### 构建
$ npm run build


### 打包脚本模块

| 模块 | 是否包含 |
|------|---------|
| es6 transpiling  | Yes |
| css splitting    | Yes |
| js minification  | Yes |
| css minification | Yes |
| sass support     | Yes |
| less support     | no  |
| source map       | Yes |
| cache busting    | No  |
| js linting       | No  |
| js unit test     | No  |

Happy coding :)
